# 가천대학교 중앙도서관 도서 DB

가천대학교 중앙도서관에서 제공받은 5/13 부스 파일럿용 도서 목록.

| 파일 | 자료실 | 권수 | 주제 비중 | 추천 우선순위 |
|---|---|---:|---|---|
| `bookcuration.xlsx` | 북큐레이션코너 (중앙도서관 1층) | 243 | AI/기술 중심 | **대출 전환 우선순위 (높음)** |
| `openlibrary.xlsx` | 프리덤광장 (오픈라이브러리) | 452 | 인문·철학·교양 | 페르소나 매칭 시 사용 |

## 컬럼

### 북큐레이션 (13컬럼)

`No.`, `등록번호`, `서명`, `저자`, `출판사`, `출판년`, `별치기호`, `청구기호`, `권호`, `복본기호`, `소장처`, `자료실`, `자료상태`

### 오픈라이브러리 (16컬럼)

위 13컬럼 + `서가`, `대출여부`, `가격`

헤더는 4행에 위치(1~3행은 출력 정보·주석). 등록번호는 도서관 내부 식별자(UEM……). **ISBN은 포함되지 않으므로** 표지 이미지·세부 메타는 Naver Books API로 서명+저자 검색해 매칭한다.

## 추천 로직 (개요)

- `bookcuration`은 AI·기술 페르소나(`vital_driver`, `editor_decider`, `focus_vital` 등)와 `진로/학습`·`경제/경영`·`과학/기술` 카테고리 선택자에게 우선 가중치
- `openlibrary`는 사색·인문 페르소나(`deep_diver`, `focused_thinker`, `balance_focus` 등)와 `인문/철학`·`시/에세이`·`소설` 카테고리 선택자에게 우선 가중치
- 최종 3권 추천은 두 출처에서 섞이도록 한다(한 출처로 100% 쏠리지 않도록 후처리 가드)
